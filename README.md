# Bubun Fashion Store

A full-stack e-commerce website built with React, Node.js, Express, MongoDB and Razorpay.

## Features
- User Authentication
- Product Catalog
- Shopping Cart
- Order Management
- Online Payment (Razorpay)
- Admin Dashboard

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Payment Gateway: Razorpay
